# Modern Air Conditioning Website

Sri Lanka's Premier Auto AC Specialists Since 1981

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- TypeScript

## Features
- Responsive modern design
- WhatsApp integration
- Service pages (Auto AC Repair, Spare Parts, Misty AC, Tours)
- Products catalogue
- Projects portfolio
- Contact form

## Getting Started

`ash
npm install
npm run dev
`

## Build

`ash
npm run build
npm start
